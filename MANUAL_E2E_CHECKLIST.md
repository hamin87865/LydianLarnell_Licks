# 수동 E2E 체크리스트

## 회원가입
1. 회원가입 진입
2. 약관 동의
3. 이메일 인증코드 발송
4. 인증코드 검증
5. 비밀번호/닉네임 입력
6. 로그인 상태 확인

## 비밀번호 재설정
1. 재설정 페이지 진입
2. 코드 발송
3. 코드 검증
4. 새 비밀번호 저장
5. 새 비밀번호 로그인

## 관리자
1. 신청 승인
2. 신청 거절
3. 상태 반영
4. 감사 로그 확인

## 업로드
1. 제목/설명/카테고리 입력
2. URL 업로드 또는 파일 업로드 중 하나만 사용
3. PDF 선택 업로드
4. 목록 반영 확인

## 결제/권한
1. prepare 성공
2. confirm 성공
3. 중복 confirm 방지
4. 구매 전 PDF 차단
5. 구매 후 PDF 허용

## 정산
1. 정산메뉴 진입
2. 월별 대상 조회
3. 지급완료 처리
4. 중복 클릭 방지
5. 스냅샷 저장 확인
